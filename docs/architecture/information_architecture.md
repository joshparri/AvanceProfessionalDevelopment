## Search design
Global search should return mixed results:
- work logs
- tasks
- clients
- knowledge entries
- playbooks
- learning items

Use:
- type labels
- short snippets
- keyboard focus
- recent searches

## Empty-state philosophy
Every empty state should show:
- what belongs here
- one example
- one clear action button
