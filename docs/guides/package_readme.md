## 🛠️ HOW TO BUILD THE APP

### Prerequisites
- Node.js (v16+) installed
- npm or yarn
- A code editor (VS Code recommended)
- 2–4 hours uninterrupted time

### Build Process

**Option 1: Use Claude Code (Easiest)**
```bash
# In terminal
claude code

# Paste the entire CLAUDE_CODE_PROMPT.md content
# Let Claude build it for you
# Follow the output; it will create a React app
```

**Option 2: Manual Build (If You Know React)**
1. `npx create-react-app avance-work-companion`
2. `npm install tailwindcss postcss autoprefixer`
3. Create the component structure shown in the prompt
4. Copy content from preloaded data sections
5. Wire up localStorage
6. Test thoroughly

### Running the App
```bash
npm start
# Opens at http://localhost:3000
# Test all 6 sections
# Make sure data persists after refresh
```

### Deploying the App (Make It Live)
**Option 1: GitHub Pages (Free)**
```bash
npm install --save-dev gh-pages
# Add to package.json: "homepage": "https://[your-username].github.io/avance-work-companion"
# npm run build
# npm run deploy
```

**Option 2: Netlify (Free, 5 mins)**
1. Build the app: `npm run build`
2. Go to netlify.com, sign up free
3. Drag-and-drop the `build/` folder
4. Get a live URL instantly

**Option 3: Just Run It Locally**
- `npm start` every time you need it
- Doesn't require internet once dependencies are installed
