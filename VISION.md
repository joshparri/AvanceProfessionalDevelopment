# Unified Vision for Avance Work Companion

## Overview
The Avance Work Companion is a private, personal web application designed to help Josh excel in his part-time IT support role at Avance Business Technology. It serves as a comprehensive tool for preparation, knowledge management, task tracking, and continuous learning in a regional MSP environment.

## Core Purpose
- **Preparation**: Equip Josh with the knowledge and checklists needed to start each shift confidently
- **Knowledge Capture**: Build a personal repository of troubleshooting playbooks, technical references, and work-related knowledge
- **Task Management**: Track tickets, shifts, and learning progress
- **Continuity**: Maintain context between part-time work days (Mondays and Wednesdays)
- **Professional Development**: Foster growth in overall IT MSP skills through structured learning, reflection, and skill-building activities
- **MSP Skills Academy**: Train practical judgement across helpdesk, endpoints, Microsoft 365, identity, Intune, networking, security, backup/DR, automation, documentation, communication, and escalation
- **Growth**: Document learning and build expertise over time

## Key Principles
- **Personal First**: Designed for one user (Josh), not a team tool
- **Practical MSP Focus**: Grounded in real regional IT support work, not idealized startup workflows
- **Low Cognitive Load**: Simple, fast interfaces that reduce stress during busy shifts
- **Offline-Capable**: Works without internet for on-site scenarios
- **Private & Secure**: Local-first storage, no sensitive data risks
- **Professional Development Emphasis**: Prioritize continuous skill building and career growth in IT MSP competencies

## Target User Journey
1. **Pre-Shift**: Review company intel, complete checklists, set priorities
2. **During Shift**: Reference tools, log tasks, capture knowledge
3. **Post-Shift**: Document learnings, update playbooks, prepare for next shift
4. **Ongoing**: Build personal expertise and confidence through professional development activities

## No Competing Visions Identified
The various documents in this repository present a consistent vision:
- A personal work companion app for part-time MSP work
- Focus on preparation, knowledge, and continuity
- Built as a local-first web app with practical features
- Emphasis on real-world applicability over generic enterprise features

Differences in documents are complementary:
- Some provide high-level strategy and app features
- Others offer detailed research on Avance and practical first-week guides
- All align on the core goal of supporting Josh's success at Avance

## Success Criteria
- Josh can prepare for shifts in under 15 minutes
- Josh can resolve common tickets independently within weeks
- Josh builds a useful personal knowledge base
- Josh maintains continuity between part-time days
- Josh demonstrates measurable growth in IT MSP skills through tracked professional development
- Josh can identify weak MSP skill areas, practise realistic ticket scenarios, produce evidence, and choose the next best action for growth
- The app feels like a trusted, practical tool (not gimmicky)

## Technical Approach
- Next.js with TypeScript for robust development
- Local storage (IndexedDB/Dexie) for privacy and offline use
- Clean UI with Tailwind CSS and shadcn/ui
- PWA features for installable local use
- Optional future Supabase sync for multi-device access

## Ethical Boundaries
- No integration with Avance systems without explicit permission
- No storage of sensitive client data
- Personal tool only, not for sharing or commercial use
- Respectful of Avance's operations and client privacy

## Evolution Path
- **MVP**: Core preparation, task tracking, and knowledge features
- **Phase 2**: Enhanced playbooks, learning tracker, search
- **Phase 3**: MSP Skills Academy with skill matrix, ticket scenarios, ticket notes practice, evidence pack, roadmap, and communication trainer
- **Future**: Optional integrations, mobile optimization, advanced analytics

This unified vision reconciles all source documents into a coherent strategy for building and using the Avance Work Companion.
